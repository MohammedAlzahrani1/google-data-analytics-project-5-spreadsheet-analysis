## Project Overview
This project is part of the Google Data Analytics Certificate (Course 2).
The goal of this project is to analyze job application data using spreadsheet
functions in order to identify monthly trends, totals, and key statistics.

## Tool Used
- Microsoft Excel

## Dataset Description
The dataset contains job application records received during the year 2023.
It includes applicant IDs, application dates, job titles, job locations,
hiring status, and application method.

## Tasks Completed
- Sorted application data chronologically
- Extracted application month using the TEXT function
- Calculated the total number of applications per month using COUNTIF
- Calculated the total number of applications for the year
- Identified the minimum and maximum monthly application counts
- Calculated the average number of applications per month
- Created a summary table to present key insights clearly

## Key Insights
- Application volume varied across the year, showing clear seasonal trends
- February recorded the lowest number of applications
- July recorded the highest number of applications
- The monthly average provides a useful benchmark for workforce planning

## Skills Demonstrated
- Spreadsheet data organization
- Excel functions (TEXT, COUNTIF, SUM, MIN, MAX, AVERAGE)
- Data summarization and analysis
- Analytical thinking and insight generation
